# Final Project 2 - MyGram App

Final Projek Kelas Kominfo. Membuat aplikasi MyGram

## Instal Project

1. Install package go
```bash
go get
```

2. Copy env
```bash
cp .env.example .env
```

3. Run App
```bash
go run main.go
```
